又到一年入学季，家有适龄儿童的家长开始焦虑了——什么时候报名？要准备什么材料？非户籍能不能上公办？政策每年都变，提前搞清楚能少很多焦虑。

本文帮你理清广州入学的基本规则和流程，不管是户籍生还是非户籍生，都能找到对应的操作路径。

## 广州入学政策总览

### 基本原则

- **就近入学**：公办学校按地段划分，住在哪就上哪附近的学校
- **人户一致优先**：孩子的户口地址和实际居住地址一致的，优先安排
- **公民同招**：公办和民办学校同步招生，不能同时被两边录取

### 时间节点（参考，以当年公布为准）

| 事项 | 时间 |
|------|------|
| 政策公布 | 4月-5月 |
| 网上报名 | 5月（一般5月上旬开始） |
| 材料审核 | 5月-6月 |
| 录取通知 | 6月-7月 |
| 民办摇号 | 6月左右 |

:::warning
报名时间窗口一般只有1-2周，错过就要等补录或下一年。关注各区教育局公众号，第一时间获取通知。
:::

## 户籍生入学

孩子有广州户籍的，走的是地段对口入学。

### 报名材料

- **户口簿**（孩子和父母的）
- **房产证**或**购房合同**（证明实际居住地）
- **预防接种证**（接种完成证明）
- **身份证**（父母双方或监护人）

### 网上报名流程

1. 登录**"广州市义务教育学校招生报名系统"**
2. 选择对应的区和学校类型（公办小学/公办初中）
3. 填写孩子和家庭信息
4. 上传材料照片
5. 提交后等待审核
6. 按通知到学校交验原件

### "人户一致"优先级

各区对"人户一致"的认定标准略有不同，但一般要求：

- 户口簿上的地址和房产证上的地址一致
- 孩子和父母（至少一方）在同一户口簿
- 实际居住在该地址

人户一致的优先级最高。如果人户不一致（比如户口在天河但住在番禺），会被统筹安排到有学位的学校，不一定是地段学校。

## 非户籍生积分入学

没有广州户籍的孩子，可以通过**积分入学**的方式申请公办学位。

### 基本条件

- 父母持有广州有效**居住证**
- 父母在广州有**合法稳定住所**和**合法稳定就业**
- 父母在广州缴纳**社保**（各区要求不同，一般需要连续缴纳一定月数）

### 积分怎么算

各区的积分指标和分值略有不同，常见指标包括：

| 指标 | 说明 |
|------|------|
| 居住证年限 | 在广州持有居住证的时间 |
| 社保年限 | 在广州缴纳社保的时间 |
| 住房情况 | 自有住房或租赁备案 |
| 学历 | 父母最高学历 |
| 纳税 | 在广州的纳税记录 |
| 志愿服务 | 社会服务时长 |

### 申请流程

1. 在规定时间内到居住地所在区的**来穗人员服务管理中心**或网上提交积分入学申请
2. 提交材料：居住证、社保证明、租赁合同/房产证、学历证书等
3. 等待积分核定和排名
4. 按积分排名和学位数量录取
5. 被录取的到学校办理注册

:::warning
积分入学的学位数量有限，不是申请了就能上。积分越高越有保障。各区竞争激烈程度不同，建议提前了解所在区往年录取的最低积分。
:::

## 幼升小 vs 小升初的区别

| 对比项 | 幼升小 | 小升初 |
|-------|--------|--------|
| 报名对象 | 年满6周岁的儿童 | 小学应届毕业生 |
| 报名系统 | 同一个招生系统 | 同一个招生系统 |
| 对口方式 | 按地段对口 | 按地段对口或电脑派位 |
| 民办摇号 | 有 | 有 |
| 特殊要求 | 需要预防接种完成证明 | 无特殊要求 |

小升初部分区采用**电脑派位**（摇号）而不是直接对口，比如越秀区的部分学校。

## 民办学校报名

如果公办学校不理想，或者非户籍生积分不够，可以考虑民办学校。

### 招生方式

- **网上报名**：在同一个招生系统中选择民办学校
- **摇号录取**：报名人数超过招生计划的，通过电脑摇号录取
- **直升**：部分九年一贯制学校有直升名额

### 志愿填报

- 可以填报**1-2所**民办学校（各区规定不同）
- 被民办学校录取后，公办学位不再保留

### 学费参考

民办学校学费差异较大，从几千到几万元一学期不等。报名前了解清楚学费和其他费用。

## 避坑提醒

- **居住登记和居住证要提前半年准备**：积分入学需要居住证，而居住证需要居住登记满半年。孩子上学前一年就要开始准备
- **社保不要断**：积分入学对社保连续缴纳有要求，断缴会影响积分
- **房产证和户口簿地址要一致**：人户一致优先级最高，地址不一致会被统筹
- **关注各区教育局公众号**：政策变化、报名时间、补录通知等第一时间发布
- **不要错过报名时间窗口**：一般只有1-2周，错过很麻烦
- **预防接种证提前办好**：幼升小需要提供预防接种完成证明，到社区卫生服务中心办理
- **民办学校学费看清楚**：除了学费还有住宿费、餐费、校服费等，算好总成本

入学是大事，提前一年准备不嫌早。关注政策变化、备齐材料、按时报名，大部分孩子都能顺利入学。
