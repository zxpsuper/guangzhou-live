在广州打拼多年，想把户口落下来？积分入户是最主流的通道之一——不看学历高低、不需要投资纳税，靠的是你在广州的综合贡献。

简单说：**满足基本条件 → 核定积分 → 排名入围 → 办理入户。**

## 申请条件

申请积分入户需要同时满足以下条件：

- 年龄**45周岁以下**
- 持有有效**广东省居住证**（广州）
- 在广州**合法稳定就业**或**合法稳定居住**
- 在广州缴纳**社会保险**满**4年**（含）以上
- 近5年内**无犯罪记录**
- 按照《广州市来穗人员积分制服务管理指标体系及分值表》计算总积分

:::warning
社保年限是硬性门槛，中间断缴会影响连续性计算。如果是灵活就业参保，也要确保连续缴纳。
:::

## 积分怎么算

广州积分入户的积分由**基础指标**和**加分指标**组成。

### 基础指标

| 指标 | 说明 |
|------|------|
| 合法稳定住所 | 在广州有自有住房、租房并办理租赁备案等 |
| 合法稳定就业 | 在广州就业并缴纳社保 |
| 年龄 | 18-40岁满分，41-45岁逐岁递减 |

### 加分指标

| 指标 | 说明 |
|------|------|
| 学历 | 本科及以上、大专、高中/中专等不同分值 |
| 技能证书 | 国家职业资格证书、专业技术资格 |
| 纳税 | 近3年在广州的个人所得税纳税额 |
| 社会服务 | 无偿献血、志愿服务、慈善捐赠等 |
| 创新创业 | 专利、高新技术企业等 |
| 住房公积金 | 缴存年限 |

### 减分指标

- 信用不良记录
- 违法犯罪记录
- 失信被执行人

具体分值以广州市来穗人员服务管理局公布的最新版本为准。建议登录**广州市来穗人员积分制服务管理信息系统**查看详细分值表。

## 如何核定积分

### 线上操作

1. 登录**广州市来穗人员积分制服务管理信息系统**
   - 网址：lsj.gz.gov.cn（或通过穗好办APP进入）
2. 注册并实名认证
3. 填写个人信息，逐项录入积分指标
4. 上传证明材料（学历证书、技能证书、纳税证明等）
5. 提交积分核定申请
6. 等待审核（一般需要15-30个工作日）

### 线下办理

到居住地所在区的**来穗人员服务管理中心**提交材料，由工作人员协助办理。

:::warning
积分核定需要时间，不要等到入户申请开始了才去核定积分。建议提前半年以上完成积分核定。
:::

## 申请流程

积分入户每年开放一次申请，大致流程如下：

### 1. 积分核定

在申请期前完成积分核定（建议提前半年）。

### 2. 网上申请

在规定申请期内，通过积分制系统提交入户申请。

### 3. 排名公示

按照总积分从高到低排名，公示入围名单。各区有不同的入户指标名额。

### 4. 办理入户手续

入围后，按通知要求到公安部门办理户口迁移手续。所需材料：

- 身份证
- 入户卡（积分入户核准卡）
- 户口簿
- 迁入地相关证明（房产证或单位集体户口同意书等）

## 历年入围分数参考

每年的入围分数线取决于当年的申请人数和入户指标数量。积分越高越稳。

影响入围的关键因素：
- 社保年限（越长越好）
- 居住证年限（越长越好）
- 年龄（18-40岁最优）
- 住房情况（自有住房加分较多）
- 学历和技能证书

## 配偶和子女随迁

积分入户成功后，配偶和未成年子女可以随迁：

- **配偶随迁**：需要结婚证、配偶身份证和户口簿
- **子女随迁**：需要出生证明、子女户口簿

随迁手续在入户时一并办理，也可以后续单独申请。

## 避坑提醒

- **积分核定要提前做**：不能临阵磨枪，核定流程需要15-30个工作日，材料不齐还会被退回
- **虚假材料后果严重**：一经查实取消申请资格，3年内不得再次申请，还可能被列入失信名单
- **社保不能断**：4年社保是硬性门槛，且断缴会影响连续性计算
- **居住证要保持有效**：居住证过期未签注会影响申请
- **每年只有一次机会**：错过申请期只能等下一年
- **各区名额不同**：选择入户区域时考虑各区的竞争激烈程度
- **房产不是必须的**：没有房产可以入单位集体户或公共集体户

积分入户是一个长期积累的过程，越早开始准备越好。在广州踏实工作、按时缴社保、保持居住证有效，积分自然会越来越高。
