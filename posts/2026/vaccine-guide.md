疫苗是预防疾病最经济有效的方式。在广州打疫苗，免费的国家免疫规划疫苗必须打，自费的二类疫苗也建议打。但很多人搞不清哪些该打、怎么预约、去哪打。

这篇文章帮你理清楚：儿童疫苗时间表、成人推荐疫苗、HPV怎么预约。

## 一类疫苗 vs 二类疫苗

| 对比项 | 一类疫苗（免费） | 二类疫苗（自费） |
|-------|----------------|----------------|
| 费用 | 国家免费 | 自费 |
| 是否必须打 | 必须 | 自愿 |
| 常见种类 | 乙肝、卡介苗、百白破、脊灰、麻腮风等 | HPV、流感、水痘、肺炎、轮状病毒等 |
| 入学查验 | 需要 | 部分需要 |

:::warning
二类疫苗虽然自费，但不代表不重要。比如水痘、流感、手足口病疫苗，都是孩子容易得的病，建议在经济允许的情况下尽量接种。
:::

## 儿童疫苗接种时间表

以下是国家免疫规划疫苗的接种时间（0-6岁）：

| 月龄/年龄 | 疫苗 |

| 2月龄 | 脊灰疫苗（第1剂） |
| 3月龄 | 脊灰疫苗（第2剂）、百白破疫苗（第1针） |
| 4月龄 | 脊灰疫苗（第3剂）、百白破疫苗（第2针） |
| 5月龄 | 百白破疫苗（第3针） |
| 6月龄 | 乙肝疫苗（第3针）、A群流脑疫苗（第1针） |
| 8月龄 | 麻腮风疫苗（第1针）、乙脑减毒活疫苗（第1针） |
| 9月龄 | A群流脑疫苗（第2针） |
| 18月龄 | 麻腮风疫苗（第2针）、百白破疫苗（第4针）、甲肝减毒活疫苗 |

## 成人推荐疫苗

大人也需要打疫苗，以下几种值得关注：

### HPV疫苗

- **二价**：预防HPV16、18型（约70%宫颈癌），9-45岁女性
- **四价**：预防HPV6、11、16、18型，20-45岁女性
- **九价**：预防9种HPV型别（约90%宫颈癌），9-45岁女性

越早打越好，最好在首次性行为之前接种。

### 流感疫苗

- 每年**9-11月**接种最佳（流感季前）
- 6月龄以上所有人都建议接种
- 尤其推荐：老人、儿童、孕妇、慢性病患者

### 带状疱疹疫苗

- 50岁以上成人建议接种
- 得过带状疱疹的人也建议接种（防止复发）

### 乙肝疫苗

- 如果乙肝表面抗体阴性，建议补种
- 可以到社区卫生服务中心查抗体再决定

## HPV疫苗预约攻略

HPV疫苗长期供不应求，九价尤其紧张。以下是广州的预约渠道：

### 预约渠道

1. **"粤苗"APP/小程序**：广州官方疫苗预约平台，大部分社区卫生服务中心的号源在这里
2. **"广州疾控"公众号**：不定期发布疫苗到货信息
3. **各社区卫生服务中心公众号**：部分中心有自己的预约入口
4. **支付宝/微信"医疗健康"**：部分私立机构的号源

### 预约技巧

- **放号时间**：各社区中心放号时间不同，提前关注公众号了解
- **提前注册**：在"粤苗"上提前注册好个人信息，放号时直接抢
- **不限户籍**：HPV疫苗不限户籍，有号就能约
- **不限居住地**：可以跨区预约，哪个区有号就去哪个区打

### 九价 vs 四价 vs 二价怎么选

- 能约到**九价**就打九价
- 九价约不到，**四价**也很好
- 四价也约不到，**二价**照样能防70%的宫颈癌
- 不要为了等九价而一直不打，越早保护越好

:::warning
HPV疫苗越早打越好。不要为了等九价而拖延，能约到什么就先打什么。保护效果远比疫苗"档次"重要。
:::

## 接种流程

不管是儿童还是成人，接种流程都差不多：

1. **预约**：通过"粤苗"APP或社区卫生服务中心公众号预约
2. **登记**：到接种点后出示预约信息、身份证（儿童带预防接种证）
3. **接种**：由医护人员接种
4. **留观**：接种后在现场**留观30分钟**，确认无异常后离开

### 接种前注意

- 感冒、发烧、腹泻期间**不能接种**
- 对疫苗成分过敏的不能接种
- 接种前如实告知医生身体状况和过敏史

### 接种后注意

- 接种部位24小时内不要沾水
- 可能出现低热、局部红肿，一般1-2天自行缓解
- 严重不适及时就医

## 跨区/异地接种

在广州接种外地疫苗本上的疫苗，或者外地来广州接种，都可以衔接：

- 带上**预防接种证**到居住地附近的社区卫生服务中心
- 工作人员会将之前的接种记录录入系统
- 后续按广州的接种安排继续接种

接种记录可以在"粤苗"APP上查询。

## 避坑提醒

- **别错过儿童疫苗的最佳接种时间**：有月龄限制，错过可能需要补种或延迟入园入学
- **HPV疫苗越早打越好**：不要纠结价数，能约到就打
- **接种后留观30分钟不能省**：万一有过敏反应，在接种点处理最安全
- **感冒发烧期间不能接种**：等好了再去，不影响后续接种计划
- **二类疫苗别因省钱而不打**：水痘、流感、手足口等疫苗能有效预防常见传染病
- **接种证要保管好**：入幼儿园和小学都要查验，丢了补办麻烦
- **预约了就按时去**：爽约可能影响后续预约信用

打疫苗是保护自己和家人最简单有效的方式。按时间表接种、及时预约HPV、每年打流感疫苗，养成习惯就好。
